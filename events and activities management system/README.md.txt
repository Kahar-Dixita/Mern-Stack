# Events and Activities Management System

A full-stack web application for managing events and activities efficiently.  
The system allows users to create, update, delete, and manage events and activities with authentication and database integration.

## 🚀 Technologies Used

### Frontend
- React.js
- Angular
- HTML5
- CSS3
- Bootstrap / Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## ✨ Features

- User Authentication
- Create Events
- Update Events
- Delete Events
- View Event Details
- Activity Management
- Responsive Design
- REST API Integration
